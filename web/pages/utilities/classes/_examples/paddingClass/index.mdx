>
  The `paddingClass` utility function can be used to construct a CSS class for
  adding padding.  If the value is true then the `pad` CSS class will be
  returned, adding a single unit (0.25rem) of padding to all sides of the element.

  If the value contains two whitespace delimited values then they are assumed
  to be vertical and horizontal padding values and a class of the form
  `pad-v-N pad-h-N` will be returned.

  If the value contains four whitespace delimited values then they are assumed
  to be padding values for the top, right, bottom and left edges. A CSS class
  of the form `pad-t-N pad-r-N pad-b-N pad-l-N` will be returned.

  In any other case the value will be prepended by `pad-`. A null or undefine
  value will return nothing.
</Example>
