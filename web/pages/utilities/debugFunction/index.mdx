# debugFunction()

<div className="large">
  This utility function can be used to create a function for sending
  debugging messages to the Javascript console.
</div>

The `debug` flag enables or disables debugging.  When it is false the `debug()`
function it returns will do nothing.  When true, the function forwards any
arguments passed to `console.log`.

The `debugPrefix` and `debugColor` allow you to define an optional prefix and
colour for that prefix for all debugging messages.  This can help to distinguish
one set of debugging messages from any others.
