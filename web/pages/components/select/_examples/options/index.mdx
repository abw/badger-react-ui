  The `options` property should be used to provide an array of options that
  the user can select from.  Each element in the array can be a string,
  numerical value or an object.

  When the `options` contains an array of objects, each object element should
  have an `id` or `value` property to uniquely identify it.  By default,
  options will be displayed using the `text`, `label` or `name` property.

  Alternately, you can define your own <CodeHashLink id="displayOption"/>
  function to display an option in the selection list and/or a
  <CodeHashLink id="displayValue"/> function to display the selected value.

  The `disabled` property can be used to mark an option
  as disabled, or you can define a <CodeHashLink id="validOption"/> function
  to implement a different strategy to identify unselectable options.
</Example>
