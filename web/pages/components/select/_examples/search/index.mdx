### `search`

The `search` property can be set to enable searching of options.  In the
simple case this can work where the `options` are all strings.  The default
search function will case-insensitively match options that contain all of
the words entered in the search input.

Try entering `aa an` in the search input in the example below. It will match
options that contain both `aa` (e.g all the <b>Aa</b>rdvarks) and `an`
(Al<b>an</b> Aardvark, <b>An</b>ne Aardvark and Bri<b>an</b> Badger).
