  The `value` property can be used to set the initial value.

  The default range is from 0 (<CodeHashLink id="min"/>) to 100
  (<CodeHashLink id="max"/>).  If a `value` is not specified then it will
  default to the midpoint of the range.

  If a specified `value` is outside of the range then it will be clamped to
  the nearest value.  If a (<CodeHashLink id="step"/>) is defined then the
  value will be quantised to the closest step value.
