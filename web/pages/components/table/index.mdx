<Warning title="Breaking Changes in v2" border>
  There were some breaking changes in the `Table` component between version
  1 and 2.  This was done to standardise some of the properties and make them
  more amenable to strict type checking with Typescript.

  The `TableV1` component is provided as an adapter to help with migration.
  It supports the same interface as the `Table` component from version 1 and
  adapts it to work with the new version 2 component.  Anywhere you have
  an import statement like this:

    `import { Table } from '@abw/badger-react-ui'`

  you should be able to change it to this:

    `import { TableV1 as Table } from '@abw/badger-react-ui'`

  and it should continue to work as before.

  The documentation for `TableV1` is [here](table-v1).
</Warning>
