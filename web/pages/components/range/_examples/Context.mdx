### Context Properties

A react context is created which provides the items listed below. Any other
properties passed in to the component will also be accessible to child
components.

Custom components can be wrapped with the `RangeConsumer` higher order
component to receive these items as properties.  Or you can use the `useRange`
hook to access them.
