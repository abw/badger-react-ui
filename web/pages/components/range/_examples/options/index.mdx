### `options`

The `options` property can be used to define an array of
discrete values for the range.  The minimum and maximum values will
be defined in terms of an index into the `options` array.
The default value for `minValue` will be 0 and the default
for `maxValue` will be one less than the length of the
`options` array.

When `options` are defined, a default `displayValue`
function will be provided which outputs the corresponding element from
the `options` array.  This is apparent in the example above.
Instead of outputting the values `0` and `25` for
the minimum and maximum ends of the range, it outputs `A` and
`Z`, which are the elements in the `options` array
at position 0 and 25 respectively.
You can define your own `displayValue` function.  It will be
passed the `value` (the index into the`options`
array) and the `options` array as arguments.
