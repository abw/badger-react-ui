  The `alignInputs` property (or `alignInput` as an alias) can be set to
  specified to set the alignment of the inputs.

  The default value is `space` which will align the minium input to the left
  edge and the maximum input to the right.  It can also be set to `center`,
  `left` or `right`.

  This option only has an effect when <CodeHashLink id="showInputs"/> is set.
