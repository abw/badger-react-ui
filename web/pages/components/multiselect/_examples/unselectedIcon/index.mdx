### `unselectedIcon`

The `unselectedIcon` property can be used to specify the icon displayed on
the right of unselected options.  There is no default value.

<Example
  Component={Name}
  code={NameSrc}
  caption="unselectedIcon"
  highlightLines="18"
  expand={false}
/>

The value can be a string corresponding to the icon `name`, as shown in the
previous example, or an object containing multiple properties.

<Example
  Component={Object}
  code={ObjectSrc}
  caption="unselectedIcon"
  highlightLines="18"
  expand={false}
/>
