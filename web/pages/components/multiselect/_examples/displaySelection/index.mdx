  Selected items are displayed using the same principle as for
  <a href="#displayoption" className="code">displayOption</a>.
  If an option is a text string then it will be displayed as it is. If it's
  an object containing a `text`, `label` or `name` property then that will be
  used as the display value.  Otherwise you can provide a `displaySelection`
  function to display the result any way you like.
