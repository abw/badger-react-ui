### `selectedIcon`

The `selectedIcon` property can be used to specify the icon displayed on
the right of selected options.  The default value is `check`.

<Example
  Component={Name}
  code={NameSrc}
  caption="selectedIcon"
  highlightLines="8"
  expand={false}
/>

The value can be a string corresponding to the icon `name`, as shown in the
previous example, or an object containing multiple properties.

<Example
  Component={Object}
  code={ObjectSrc}
  caption="selectedIcon"
  highlightLines="8-12"
  expand={false}
/>
