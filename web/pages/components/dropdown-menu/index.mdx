## CSS Classes

### `optionClass`

Used to set the CSS class added to each dropdown menu item. The default
value is `item no-hover`.  The `no-hover` class suppresses the hover styles
that would usually be applied to menu items.  Instead the `active` class is
added to the active item, selected either by hovering over it or by keyboard
navigation.

### `separatorClass`

Used to set the CSS class added to the dropdown menu separator.
The default value is `separator`.
