### `options`

The `options` property should be used to define an array of options to
display in the menu. For simple cases, each element in the array can be a
text string or numerical value.

<Example
  Component={Values}
  code={ValuesSrc}
  caption="options"
  highlightLines="4-9"
  expand={false}
  undent={2}
/>

An element in the `options` array can also be an object. It should contain
a `text` property which can be a string or JSX fragment. A `className` can
be included to add a CSS class to the option.

The content for an element is rendered using <WithIconsLink/> allowing you to
include `iconLeft` and `iconRight` properties to add icons to either side,
along with `iconLeftClass` and `iconRightClass` to set CSS classes for those
icons.

<Example
  Component={Complex}
  code={ComplexSrc}
  caption="options"
  highlightLines="4-26"
  expand={false}
  undent={2}
/>

An object in the `options` array can have a `disabled` property set to
`true` to disable the option.

<Example
  Component={Disabled}
  code={DisabledSrc}
  caption="options"
  highlightLines="7,14,21"
  expand={false}
  undent={2}
/>

An object in the `options` array can have a `separator` property set to
`true` to create a separator.

<Example
  Component={Separators}
  code={SepsSrc}
  caption="options"
  highlightLines="8,12"
  expand={false}
  undent={2}
/>

An object in the `options` array can have a `heading` property to create a
section heading.
