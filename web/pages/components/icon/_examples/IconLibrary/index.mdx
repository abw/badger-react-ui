### `IconLibrary`

The default set of icons is defined as `IconLibrary`.
The `addIcon()` and `addIcons()` function are
provided for adding icons to the library.

<div className="grid-fill gap-4 mar-t-8">
  { Object.keys(IconLibrary.icons).map(
    name =>
      <div
        key={name}
        className="text-center bgc-100 bgd-0 fgc-10 fgd-90 pad-4 shadow-2"
      >
        <Icon name={name} className="x3"/>
        <div className="small mar-t-4">{name}</div>
      </div>
  )}
</div>

### Adding Icons

You can use the `addIcon()` and `addIcons()` functions to add icons to the
library.  If you don't want to update the standard library then you can create
your own copy of the `IconLibrary` (e.g. by using `structuredClone()` as shown
in this example) and then pass that as the third argument to `addIcon()` or
second argument to `addIcons`.

You should then pass your icon library to the `Icon` component using the
`library` property.

You can use a <Link to="/theming/theme-provider">Theme Provider</Link> to
automatically set the `library` property to your custom library
to save you from having to specify it every time you use the `Icon`
component.
