  The `padding` property (or `pad` for short) can be used to add padding to
  the element.  It can be one, two or fours numbers in units of 0.25em.
  A single number is used for all four side.  Two numbers can be used for
  vertical and horizontal padding.  Four numbers can be used for the top,
  right, bottom and left padding.  When used as a boolean flag it defaults to
  1 unit.
